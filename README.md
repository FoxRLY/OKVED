# Игра "Найди ОКВЭД по номеру телефона"
Тестовое задание
## Необходимые зависимости:
- python 3.13
- uv (https://docs.astral.sh/uv/getting-started/installation/)
## Как запускать?
```uv run main.py +79999999999```
## Пример работы:
```
uv run main.py +79999999910

Ищем ОКВЭД по номеру +79999999910...
ОКВЭД найден!

Телефон: +79999999910
ОКВЭД: 91.0 | Деятельность библиотек, архивов, музеев и прочих объектов культуры
Длина совпадения: 3
```
